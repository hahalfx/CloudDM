## 1.4 云边交互通信协议

• 为实現高效通信，推荐采用 JSON-RPC 作为通信协议，并优先选择 WebSocket 作为底层传输协议。协议应使用清晰的JSON格式封装请求和响应，且具备良好的通用性和易用性，确保云边端稳定的数据传输。
